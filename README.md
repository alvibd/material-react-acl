# Material React Access Control Logic

Easy Installation
-----------------
**Project Setup Guide**

[Dockerizing a React App](https://mherman.org/blog/dockerizing-a-react-app/#react-router-and-nginx)

**Requirements**
- docker
- docker-compose

**Installation Step**
- run : `$docker-compose up -d`
- to stop run: `$ docker-compose down --remove-orphans`
